# TP1 – Deep Learning : Conception, Entraînement, Suivi et Déploiement d’un Modèle MNIST

##  Description du projet
Ce dépôt contient les travaux réalisés dans le cadre du **TP1 de Deep Learning** à l’ENSPY, dans l’unité d’enseignement *Création d’une IA 2*.  
L’objectif du TP est de construire, entraîner, suivre et déployer un modèle de Deep Learning pour la classification des chiffres manuscrits du dataset MNIST.

Le projet couvre :
- la construction d’un réseau de neurones fully-connected avec **Keras/TensorFlow** ;
- le suivi d’expérimentations avec **MLflow** ;
- la création d’une API Flask pour servir le modèle ;
- la conteneurisation de l’application avec **Docker** ;
- la mise en place d’un pipeline de **CI/CD** (concepts).
